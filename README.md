# U-Booty i Niszczyciele — prototyp

Projekt jest podzielony na trzy pliki:

- `index.html` — struktura strony,
- `style.css` — wygląd interfejsu,
- `game.js` — cała logika gry.

## Uruchomienie na GitHub Pages

Wrzuć wszystkie trzy pliki do głównego katalogu repozytorium i włącz GitHub Pages dla gałęzi `main`, katalog `/(root)`.

## Sterowanie

- `W/S` — rozkaz prędkości
- `A/D` — ster +/- 5°
- `Z` — ster zero
- `000-359 + Enter` — kurs okrętu
- `G` — przełącza tryb wpisywania: kurs / kąt celowania
- `F` — tryb automatycznego kierunku dział
- `<` `>` — płynny obrót dział
- `O/L` — płynna zmiana zasięgu
- `[` `]` — kierunek sonaru
- `P` lub przycisk — sonar
- `Spacja` — strzał
- kółko myszy — zoom
- LPM + przeciągnięcie — przesunięcie mapy
- podwójny klik — wycentruj kamerę
